HTML: Kerangka Web

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun semua konten di halaman web. Anggapannya HTML adalah arsitek yang membangun fondasi, dinding, dan atap sebuah rumah. Tanpa HTML, tidak ada struktur sama sekali, hanya tumpukan material mentah.

Sejarah Singkat HTML

Diciptakan oleh Tim Berners-Lee pada tahun 1991, HTML dimulai sebagai cara sederhana untuk berbagi dokumen ilmiah. HTML 2.0 (1995) menjadi standar pertama. HTML 4.01 (1999) menjadi versi yang sangat populer dan dominan selama bertahun-tahun. Versi modern yang kita gunakan saat ini adalah HTML5 (2014), yang memperkenalkan banyak fitur canggih seperti tag <video>, <audio>, dan elemen semantik seperti <article> dan <header> yang membuat kode lebih bermakna.

Struktur Dasar & Contoh

<!DOCTYPE html>
<html lang="id">
  <head>
    <title>Judul Halaman Saya</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah website pertama saya.</p>
  </body>
</html>

CSS: Design Web

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan menata tampilan halaman HTML. Jika HTML adalah kerangka rumah, CSS adalah cat, wallpaper, perabotan, dan semua dekorasi yang membuatnya indah dan nyaman.

Jenis-Jenis CSS & Contohnya

  1. Inline CSS: Style diterapkan langsung ke satu tag HTML. 
    <h1 style="color: blue;">Judul Biru</h1>
  2. Internal CSS: Style ditulis di dalam tag <style> di bagian <head>. 
    
<head>
  <style>
    p {
      color: green;
      font-size: 18px;
    }
  </style>
</head>
  3. External CSS: Style ditulis di file terpisah (misal: style.css) dan dihubungkan ke HTML. Ini adalah praktik terbaik karena memisahkan struktur dan desain. 
    
/* Di dalam file style.css */
body {
  background-color: #f0f0f0;
}

JavaScript

JavaScript (JS) adalah bahasa pemrograman yang membuat website menjadi interaktif. Ibaratnya JavaScript adalah sistem kelistrikan dan semua perangkat elektronik di dalam rumah.

Contoh & Hasil

Berikut contoh JS yang mengubah teks ketika tombol di bawahnya diklik.

Kode JavaScript: 

const tombol = document.getElementById('tombolUbah');
const teks = document.getElementById('teksAjaib');

tombol.addEventListener('click', () => {
  teks.textContent = 'Teks ini telah diubah!';
});
Hasilnya (Coba Klik):

Ini adalah teks awal.

Tailwind CSS

Mirip dengan CSS, Tailwind CSS adalah sebuah framework CSS yang bekerja dengan cara berbeda.Tailwind menggunakan class-class "utilitas" yang sudah disediakan oleh Tailwind langsung di dalam HTML.

Kelebihan Dibanding CSS Biasa & Contoh

  • Tidak Perlu Menamai Class:Kita tidak perlu pusing memikirkan nama class yang baik.
  • Pengembangan Super Cepat: Desain dibangun langsung di HTML, mengurangi waktu bolak-balik antar file.
  • Desain Konsisten: Menggunakan skala desain yang sudah ditentukan (untuk warna, spasi, ukuran font) sehingga hasilnya konsisten.
CSS Biasa: 

<!-- HTML -->
<button class="tombol-primer">Klik Saya</button>

/* CSS */
.tombol-primer {
  background-color: #3b82f6;
  color: white;
  padding: 8px 16px;
  border-radius: 6px;
}
Dengan Tailwind CSS: 

<!-- Semua di HTML -->
<button class="bg-blue-500 text-white py-2 px-4 rounded-md">
  Klik Saya
</button>

Mari buat web sederhana di mana pengguna bisa memasukkan data melalui form, dan data tersebut akan muncul di dalam tabel secara dinamis.

Langkah 1: Membuat Struktur dengan HTML

Pertama, siapkan kerangka website. Buat file bernama index.html (untuk tutorial ini) dan isi dengan kode berikut. Kode ini berisi form input dan tabel kosong.


<!DOCTYPE html>
<html lang="id">
<head>
  <title>Form ke Tabel</title>
</head>
<body>
  <div class="container">
    <h1>Daftar Peserta</h1>
    <form id="form-peserta">
      <label for="nama">Nama:</label>
      <input type="text" id="nama" required>
      <label for="email">Email:</label>
      <input type="email" id="email" required>
      <button type="submit">Tambah ke Tabel</button>
    </form>
    <table>
      <thead>
        <tr><th>Nama</th><th>Email</th></tr>
      </thead>
      <tbody id="data-tabel"></tbody>
    </table>
  </div>
</body>
</html>

Langkah 2: Menghubungkan CSS ke HTML

Sekarang, buat file style.css. Lalu hubungkan ke HTML dengan menambahkan <link rel="stylesheet" href="style.css"> di dalam <head> HTML Anda.

<!DOCTYPE html>
<html lang="id">
<head>
<title>Form ke Tabel</title>
<link rel="stylesheet" href="style.css">
</head>

Isi style.css dengan style yang diinginkan.

Langkah 4: Menghubungkan JavaScript ke HTML

Agar fungsionalitas berjalan, kita perlu menghubungkan file JavaScript. Buat file script.js dan tambahkan tag <script> tepat sebelum tag penutup </body> di file HTML Anda. Ini memastikan HTML dimuat terlebih dahulu sebelum script dijalankan.

    <!-- ...konten body lainnya... -->
    <script src="script.js"></script>
  </body>
</html>

Langkah 5: Menulis Logika dengan JavaScript

Isi file script.js Anda dengan logika untuk mengambil data dari form dan menambahkannya ke tabel.


const form = document.getElementById('form-peserta');
const inputNama = document.getElementById('nama');
const inputEmail = document.getElementById('email');
const tabelBody = document.getElementById('data-tabel');

form.addEventListener('submit', function(event) {
  event.preventDefault();

    const nama = inputNama.value;
    const email = inputEmail.value;

    const barisBaru = document.createElement('tr');
    const selNama = document.createElement('td');
    const selEmail = document.createElement('td');

    selNama.textContent = nama;
    selEmail.textContent = email;

    barisBaru.appendChild(selNama);
    barisBaru.appendChild(selEmail);

    tabelBody.appendChild(barisBaru);

    form.reset();
});

Hasil Akhir Interaktif

Jika semua file digabungkan, hasilnya akan seperti di bawah ini. Coba isi form dan klik tombol "Tambah ke Tabel" untuk melihat data Anda muncul di tabel secara langsung!
